# maven

## 3.6.3

```sh
mvn -v
```

### Help

```sh
mvn help:describe -Dplugin=help
mvn help:describe -Dplugin=help -Ddetail
```

### Info

```sh
mvn help:effective-pom
mvn help:effective-settings
```

### Build

```sh
mvn compile
mvn test
mvn verify
mvn package
mvn package -DskipTests
mvn clean
```
