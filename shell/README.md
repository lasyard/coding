# shell

1. `basic_mediainfo.sh`  
   用 `mediainfo` 显示多媒体文件信息并用 `jq` 提取主要信息成一行文本
1. `centos_yum_install.sh`  
   CentOS 上用 yum 安装常用软件
1. `datify`  
   从文件名中提取时间戳并以此时间命名文件
1. `hz_count.sh`  
   统计汉字个数
1. `install_java_bin.sh`  
   安装 java Binary 包
1. `lasync.sh`  
   使用 rsync 进行同步
1. `macos_brew_install.sh`  
   macOS 上用 homebrew 安装常用软件
1. `profile.sh`  
   Shell 登录脚本
1. `venc.sh`  
   使用 `ffmpeg` 编码视频为 mp4
