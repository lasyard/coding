# shell

1. `basic_mediainfo.sh`  
   用 `mediainfo` 显示多媒体文件信息并用 `jq` 提取主要信息成一行文本
2. `centos_yum_install.sh`  
   CentOS 上用 yum 安装常用软件
3. `install_java_bin.sh`  
   安装 java Binary 包
4. `lasync.sh`  
   使用 rsync 进行同步
5. `macos_brew_install.sh`  
   macOS 上用 homebrew 安装常用软件
6. `profile.sh`  
   Shell 登录脚本
7. `venc.sh`  
   使用 `ffmpeg` 编码视频为 mp4
